# Content Repurposing Agent

An AI-powered automation that converts YouTube videos into LinkedIn posts automatically.

## How it works
1. Takes a YouTube video URL
2. Extracts the transcript via RapidAPI
3. Sends transcript to Groq AI (Llama 3.3)
4. Generates a LinkedIn post
5. Saves output to Google Sheets

## Built with
- n8n
- RapidAPI (YouTube Transcript)
- Groq AI (Llama 3.3-70b)
- Google Sheets
